# PC Recommendation Engine

An AI-powered recommendation system that analyses user preferences and suggests optimal PC configurations.

## Tech Stack
Python · Alteryx · ETL Pipeline · Machine Learning · Data Validation · REST API

## Features
- Processed 200+ user datasets to identify purchasing patterns
- Full ETL pipeline: data extraction, cleaning, transformation, feature engineering
- Improved recommendation accuracy by 25% through iterative validation
- Interactive dashboard for non-technical stakeholders
